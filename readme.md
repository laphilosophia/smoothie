## yet another smooth scroll thing.

tiny (1KB), simple, bla bla plugin for smooth scroll stuffs.

[Bon App√©tit](https://www.youtube.com/watch?v=dPI-mRFEIH0)


### Usage
````javascript
    // @usage
    const _scroll = new smoothscroll({
        behavior: 'smooth',
        top: 0,
        left: 0
    })
````

### Calling
````javascript
    _scroll.scrollTo()

    // or

    _scroll.scrollBy()
````
